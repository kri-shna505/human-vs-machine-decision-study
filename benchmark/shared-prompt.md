# Frozen four-application benchmark prompt

Status: **not frozen yet**

A real comparison is invalid until every application is generated or evaluated
against the same frozen requirements.

The final prompt must include, at minimum:

1. Anonymous consent and participant registration.
2. Three decision-making scenarios with answer and confidence collection.
3. Refresh recovery and duplicate-submission protection.
4. A backend API and persistent relational database.
5. A supervisor-only presentation journey isolated from research results.
6. A comparative-analysis screen with honest data labeling.
7. Responsive design and keyboard-usable controls.
8. Automated tests and reproducible deployment instructions.
9. Security, privacy, and secrets-handling requirements.
10. A fixed time budget and a written rule for allowed manual corrections.

- export or preserve source code for reproducibility
